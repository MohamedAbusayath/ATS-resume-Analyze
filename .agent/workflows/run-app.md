---
description: Run the ATS Resume Checker (Backend and Frontend)
---

To run the ATS Resume Checker application, follow these steps:

1. **Start the Backend:**
   - Open a terminal in the root directory.
   - Run the following command:
     ```bash
     mvn spring-boot:run
     ```

2. **Start the Frontend:**
   - Open a new terminal in the `frontend` directory.
   - Run the following command:
     ```bash
     npm run dev
     ```

3. **Access the App:**
   - Open your browser and navigate to `http://localhost:5173`.
